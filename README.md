# Stock Manager
----------------------

A Qt5 GUI software for managing stocks

You need [Conan Package Manager](https://conan.io/) and CMake 2.8.11 or higher

## Build instructions

Clone this repo
```sh
git clone https://github.com/krivahtoo/stockmanager.git
```

### Build Dependencies

- [conan](https://conan.io/)
- cmake

### Linux
```sh
./build.sh
```

### Windows
```bat
.\build
```
